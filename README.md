# English Roadmap

> My journey from A2 to fluent English.

## 🎯 Goal

This repository records my English learning journey from **A2** toward
**B1, B2, and beyond**.

My goals are:

-   Improve my speaking
-   Improve my listening
-   Improve my reading
-   Think in English instead of translating from Chinese
-   Speak naturally with people from different countries
-   Find a better job with stronger English skills

This repository is **not** an English textbook.

It is a record of my progress, challenges, reflections, and growth.

## 📝 Journal

Every English lesson creates one journal entry.

Each journal records:

-   What I practiced
-   What I learned
-   My biggest mistake
-   My biggest improvement
-   My reflection
-   My next goal

The journal is meant to capture my journey---not perfect English.

------------------------------------------------------------------------

## 📚 Handbook

The handbook only contains knowledge worth keeping for years.

Examples:

-   Grammar
-   Vocabulary
-   Native expressions
-   Common mistakes
-   Speaking tips

It is updated only when something is truly valuable.

## 📈 Monthly Progress

At the end of each month, I review my progress.

Examples:

-   Speaking
-   Listening
-   Reading
-   Writing
-   Confidence
